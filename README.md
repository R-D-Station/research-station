## 🛠️ Как внести свой вклад

1. **Fork & Clone:** Создайте форк данного репозитория и клонируйте его на своей машине.
2. **Engine:** Установите [Godot 4.5](https://godotengine.org/download).
3. **Import:** Откройте Godot Project Manager и импортируйте `project.godot`.
4. **Submit:** Внесите и отправьте все свои изменения в форк, а затем просто создайте PR.
